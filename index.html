My-first-blog-我是如何创建属于自己的博客的

我的第一篇博客

简介：个人第一次写博客，简单记录一下，关于如何搭建博客，如何搭建自己的服务器，如何连接GitHub等等

第一步：搭建环境

  • 第一步：搭建git环境，百度谷歌好多，搜搜就行，别忘了免密登陆github

  • 第二步：搭建nodejs环境（如何安装cnpm是个大坑，我研究了好长时间，参考博客https://blog.csdn.net/lhy2199/article/details/81710645）

    • 注意：一定要注意设置系统变量！！！

    • 在NodeJs下建立“node_global”及“node_cache”两个文件夹

    • npm config set prefix “c:\Program Files\nodejs\node_global” #修改npm全局目录

    • npm config set cache “c:\Program Files\nodejs\node_cache”

    • 下面这一步非常关键，我们需要设置系统变量。进入我的电脑→属性→高级→环境变量。在系统变量下点击path，编辑系统变量，输入“C:\Program Files\nodejs\node_global”！这一步的目的是为了让win10下的命令框窗口，可以输入类似于hexo，vue，cnpm等命令，以免出现command not found尴尬的情况
      ##第二步：创建自己的博客

  • 第一步 安装hexo 详细参考（http://www.atangbiji.com/2019/12/22/buildHexoBlog/）

  • 主要是这三个命令

    • $ hexo init myblogs
    • $ cd myblogs
    • $ npm install
      由于网速原因，多下几次…我下了将近10次才成功…

  • 第二步：创建自己的博客

    • hexo n “My first blog.”
    • 进入/source/_posts 可以看到自己创建的markdown文件
    • 输入命令 hexo s
    • 在Chrome中输入 localhost：4000 可以看见自己的博客啦。

后续关于如何连接GitHub，如何连接云服务器，域名解析等等的操作还会分享。
